# Palette's Journal

## 2025-05-15 - [Dynamic Context & Empty States]
**Learning:** Screen reader users benefit significantly from dynamic ARIA labels on state-dependent buttons (e.g., "Help Level: Beginner") rather than generic labels. Additionally, inline search filters without "No results found" feedback create "dead ends" that confuse both sighted and non-sighted users.
**Action:** Always include the current state in ARIA labels for toggle/cycle buttons and implement a clear empty state message for all searchable lists or dropdowns.
